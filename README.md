# Evaluación 1: Infraestructura como Código II

## Propósito General y Objetivos
Este repositorio contiene la evaluación práctica para automatizar análisis de calidad y seguridad.

## Instrucciones básicas de uso
1. Clonar el repositorio.
2. Ejecutar `terraform init`.
3. Ejecutar `terraform plan`.

## Definición del código Terraform
Este proyecto despliega una VPC con subredes y una máquina virtual EC2 en AWS utilizando Terraform.
